説明

就職活動用に制作したアクションゲームです。
プレーヤーはトレジャーハンターのキャラクターを操作し、山頂の洞窟にある宝石を目指します。
プレーヤー・敵、共にヒットポイントバーが設定されており、攻撃によりそれが0になる事で死亡します。
敵は弱敵と強敵(ボス)の2種類が存在し、強敵は弱敵をすべて倒し奥の洞窟に入ると出現します。
弱敵を倒すとアイテムチェストが出現し開くとプレイヤーの能力をアップさせるアイテムが出現します。
アイテム一覧
①ハート・・・体力を回復します
②弓矢・・・攻撃速度を上昇させます
③スニーカー・・・移動速度を上昇させます

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

作品アピールポイント

このゲームでは弓をフックショットのように使い通常のジャンプではいけないところに上ったり敵の攻撃を避けることができます。
それによって「スタイリッシュさ」をプレイヤーに感じてもらいます。
壁や崖に向かって走ると自動で壁を登ったりジャンプしたりします。直観的に移動できるようにしています。


~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

技術面説明

ゲームジャムにて学んだオブジェクトの管理システムである「タスクシステム」を使用し、ゲーム内に登場する大量のオブジェクトを纏めて管理しています。
別のゲームを作っていた時に学んだステートマシンを使い、コードの管理は複雑化させずに、プレイヤーの行動を複雑化させることに成功しました。
